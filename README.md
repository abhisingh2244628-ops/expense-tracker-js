# 💰 Expense Tracker (JavaScript)

A simple and responsive Expense Tracker web application built using HTML, CSS, and Vanilla JavaScript.

## 🚀 Features
- Add income and expense transactions
- Automatically calculate total balance
- Display total income and total expenses
- Delete transactions
- Data stored in LocalStorage
- Responsive UI

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API

## 📚 What I Learned
- DOM Manipulation
- Event Handling
- Array Methods
- LocalStorage
- Basic UI/UX structuring

## 🎯 Future Improvements
- Add categories
- Add charts (using Chart.js)
- Monthly filtering
